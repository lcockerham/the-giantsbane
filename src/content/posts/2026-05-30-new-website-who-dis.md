If you've visited before, you may notice things look a little different around here. The Giantsbane has moved off Blogger and onto a proper home at [thegiantsbane.com](https://thegiantsbane.com).

## Why the change?

Because blogger is so 2000 and late, and with Claude Code I can custom build a new site and migrate my old content in a reasonable amount of time. If you are interested in the technical details check out the [github repo](https://github.com/lcockerham/the-giantsbane).

## What's the same

Claude Code migrated all my posts, but I had to do quite a bit of clean up, so you may see some issues here and there, but the writing is still all me. 

## What's new

A couple things I've added:

- **RSS feed** — if you use a feed reader, point it at `/rss.xml` and you'll get new posts automatically.
- **Email subscription** — there's a subscribe form in the sidebar if you'd rather get new posts in your inbox.
- **Feedback via email** - no more comments, but you can send me feedback via email. 


## What's coming

A few things are on my mind - my Age of Worms campaign has been on hiatus as I have been quite busy. But I have almost finished running the fourth adventure and will have some comment coming for it. I also want to write some about using LLMS to help prep dnd, especially using a LLMs-as-wikis approach. I want to do a review of the latest WotC Forgotten Realms books as well. Hit me up with the feedback feature to let me know what you are interested in!
